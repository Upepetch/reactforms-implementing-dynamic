# React Job Management App

A dynamic job tracking application built with React. Supports adding, deleting, and updating job status across columns.

## Features
- Add new jobs with default status
- Drag-and-drop jobs between columns
- Update job status via dropdown
- Delete jobs with a single click
- Animated transitions between columns

## Getting Started

##```bash
npm install
npm start

Folder Structure
src/component/: Reusable UI components

src/images/: Icons and illustrations

App.js: Main logic and state management
